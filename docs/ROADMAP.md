# Roadmap

## Phase 1: MVP (1-2 εβδομάδες)

1. Flutter app με απλό UI για contacts/messages.
2. Audio capture + upload.
3. Backend APIs για contacts/messages.
4. Speech-to-text pipeline.
5. Outbound send via Viber Bot + Messenger API.
6. Inbound webhooks και προβολή απαντήσεων.

## Phase 2: Security Hardening

1. Secrets manager + token rotation.
2. Webhook signature verification (strict).
3. Rate limiting ανά επαφή/συσκευή.
4. Audit trail + alerts + kill switch.
5. Encrypted object storage για audio.

## Phase 3: Grandma UX Optimization

1. Μεγαλύτερα controls, voice prompts.
2. Offline/resume support.
3. Auto-retry sending.
4. Field tests με πραγματικά σενάρια.

## Phase 4: Production Ops

1. CI/CD.
2. Monitoring dashboards.
3. Backup + disaster recovery.
4. Compliance review και bot policy checks.
